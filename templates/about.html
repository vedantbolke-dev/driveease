About DriveEase

DriveEase is a fully functional car rental management system developed as a final year project for B.Sc. Computer Science at Shri Dnyaneshwar Mahavidyalaya, Newasa. It demonstrates real-world web development using Python, Django, MySQL and Bootstrap 5.
